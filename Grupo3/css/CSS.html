Background-color

bagraund-color:define el fondo de un parrafo de un titulo u otros elementos de la pagina web.
Estos pueden ser transparentes por defecto, como tambie pueden tener colores solidos, imagenes o esfumados. podemos combinar propiedades de background, backgroun-Color, background-image

Background-image

Este define una o varias imagenes de fondo de un elemento Para la iamgen de fondo se usa un "background-image: url ('.gif') las demas se usan con una ruta (.png)

Background-repeat

Define si la imagen debe ser repetida, dependiendo los comandos asignados repeat, repeat-x, repeat-y, no-repeat.. cada uno define si debe repetirse si no lo debe hacer o si de que manera hacerlo

Background-position

Define la posicion de la imagen, dependiendo los comandos asignados top-left, top-center, top-right, center-left, center-center, center-right, bottom-left, bottom-center, bottom-right.. cada uno define la posicion como debe estar, como debe estar en la pag web

Background-attachment

Define la si la imagen de fondo es scroll o fixed, es decir, si se debe mover con el texto o elementos de la pagina o no debe hacerlo. dependiendo los comandos asignados fixed,scroll..

Background-origin

Define el origen de la imagen, es decir,La propiedad background-origin especifica el área de origen de un fondo o imagen en determinada caja, dependiendo los comandos asignados border-box, padding-box, conten-box..

Background-size

Define el tamaño de la imagen, dependiendo los comandos asignados auto,cover,contain..cada uno define la posicion como debe estar, como debe estar en la pag web

Background-clip

Define si el fondo debe ser acortado, dependiendo la caja border-box,padding-box,conten-box,no-clip..por ejemplo: 1¿Qué ocurre si una imagen es más grande que su elemento contenedor? - La propiedad de clip le permite especificar un rectángulo para recortar un elemento con posición absoluta. El rectángulo se especifica como cuatro coordenadas, todo ello desde la esquina superior izquierda del elemento a ser cortada.
